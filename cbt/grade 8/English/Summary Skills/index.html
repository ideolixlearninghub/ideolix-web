Precision

Selecting exactly the right words to convey meaning accurately without ambiguity or vagueness.

Key Features:

  • Specific word choice
  • Avoiding vague language
  • Accurate representation
  • No exaggeration
  • Factually correct
  • Clear boundaries

Clarity

Making information understandable by organizing ideas logically and using straightforward language.

Key Features:

  • Simple sentence structure
  • Logical organization
  • Clear transitions
  • Direct expression
  • Avoiding jargon
  • Readable formatting

Accuracy

Maintaining factual correctness while condensing information, preserving the original meaning.

Key Features:

  • Fact preservation
  • No distortion
  • Proper context
  • Source integrity
  • Objective representation
  • Balanced perspective
üìâ

Condensing

Reducing content to essential information while maintaining coherence and completeness.

Key Features:

  • Eliminate repetition
  • Remove examples
  • Combine related ideas
  • Focus on main points
  • Use generalizations
  • Reduce word count
